ВАЖНО!!! Перед установкой любых приложений настоятельно рекомендуется обновлять информацию о репозитории!!!:

# opkg update
# opkg upgrade

Для установки приложения из репозитория Entware используем команду:

# opkg install { app_name }

Эта команда устанавливает приложение с его зависимостями.

 ВАЖНО: Никогда не удаляйте ничего самостоятельно из каталога /opt!!! Приложения удаляются с помощью команды:

# opkg remove { app_name }

Например, для установки Midnight Commander, выполняем команду:

# opkg update
# opkg install mc

Теперь имеем установленный Midnight Commander.

Используйте opkg list для просмотра доступных в репозитории приложений.
Для получения дополнительной информации используется команда opkg --help.
