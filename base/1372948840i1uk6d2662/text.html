печать через gsprinter

1.       Установить gs905w32 в папку c:\gs\

2.       Распаковать redmon в папку c:\gs\redmon и установить оттуда.

3.       Расшарить реальный принтер, указать в качестве его сетевого имени gsprinter, в дополнительных настройках выбрать галочку – печатать прямо на принтер.

4.       Создать виртуальный локальный принтер, в качестве порта выбрав redirected port, далее ничего не менять (RPT:1). Вручную указать модель принтера HP Color Laserjet 4550 PS, имя указать GS PS Printer и не выбирать использовать по умолчанию, затем далее – далее (расшарить по сети) и не печатать пробную страницу.

5.       Скопировать файл в папку c:\gs\mswinpr2.txt.

6.       Зайти в настройки порта RPT1:

a)      В «Redirect this port to the program» вписываем  «C:\gs\bin\gswin32c.exe»

b)      В «Arguments for this program are» вписываем «@c:\gs\mswinpr2.txt –sOutputFile=”\\spool\gsprinter” -»

c)       В output выбрать «program handles output»

d)      Run «Hidden»

7.       Запустить gsprinter.reg и добавить в реестр.

ВСЕ!!!
