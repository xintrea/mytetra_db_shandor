Для запуска АВК из под учётной записи с ограниченными правами можно воспользоваться следующим способом --

- создать ярлык на рабочем столе пользователя (при установке ярлык создаётся на рабочем столе того кто устанавливал -- администратора)

-- открыть свойства ярлыка и в поле объект прописать следующее:

runas /env /savecred /user:Администратор C:\AVK3\EXE\avk.exe,

здесь Администратор -- пользователь с правами администратора от имени которого мы будем запускать программу, C:\AVK3\EXE\avk.exe -- путь к исполняемому файлу АВК

-- после этого на этот ярлык нужно будет назначить иконку из файла avk.exe (можно и не назначать)

-- при первом запуске этого ярлыка откроется окно с предложением ввести пароль пользователя пользователя, которого мы прописали в параметре /user:, после ввода система его запомнит и больше спрашивать не будет.
